# Gold Advisor Midnight ‚Äî Claude Context

## Project

## Branch & Commit Conventions
- **New features** ‚Üí branch from `discord` (`feat/<name>`) ‚Üí merge back to `discord`
- **Bug fixes** ‚Üí branch from `main` (`fix/<name>`) ‚Üí merge to `main`, then also to `discord`
- **Commit style:** `feat(scope): description` / `fix(scope): description` / `chore(data): description`
- Bump `ADDON_VERSION` in `Constants.lua` **and** `## Version:` in the TOC together ‚Äî never one without the other
- Bug fixes: increment Z only (1.6.0 ‚Üí 1.6.1). Z has no digit cap.
- Feature releases: increment Y, reset Z to 0 (1.6.x ‚Üí 1.7.0). Y max is 2 digits (99).
- Major (X) bump: only when explicitly requested.

## DB Migration
- Increment `DATA_VERSION` in `Constants.lua` when the SavedVariables schema changes (new fields, renamed keys, removed keys)
- A version bump wipes the price cache on the player's next login ‚Äî do this intentionally, not casually

## Off-Limits / Do Not Read Unless Asked
- `references/` ‚Äî third-party addon source (Ace3, ARP, broken legacy versions) and spreadsheets. Read only if directly relevant to the task.
- `references/Working Addons/` and `references/WIP:broken addons/` ‚Äî historical/reference code, not our codebase.
- `references/Wow api Docs/` ‚Äî large Blizzard API reference tree. Look up specific files only if needed.
- `Data/*Generated.lua` ‚Äî auto-generated by `tools/generate_workbook_data.py`. Do not edit manually.
- `releases/` ‚Äî built zip artifacts, not committed.
- `build/` ‚Äî legacy/archived scripts.
- `.venv/` ‚Äî Python virtual environment.
- `memory/` ‚Äî Claude Code memory files, not addon code.

## Data / Generation
- `source/GoldAdvisorMidnight/Data/` files are generated Lua. Regenerate them from the spreadsheet instead of editing them by hand.
- `references/Spreadsheet/*.xlsx` are the source of truth for item data. Do not attempt to read them directly.
- `tools/compare_strats.py` ‚Äî verifies `StratsGenerated.lua` matches the spreadsheet. Must show 0 mismatches before any release.
- `tools/verify_stat_scaling.py` ‚Äî verifies crafting stat scaling calculations against expected values.
- `tools/manual_strats.json` ‚Äî supplemental strategy entries not yet in the spreadsheet; regenerated and appended to `StratsGenerated.lua` by the generator with a `-- MANUAL:` marker.

## Release Workflow
- Version lives in **two places**: `## Version:` in `GoldAdvisorMidnight.toc` and `ADDON_VERSION` in `Constants.lua` ‚Äî bump both.
- `Sync_Addon.command` ‚Äî rsync source into local WoW AddOns directory for testing.
- `Release_Discord.command` ‚Äî plain zip + GitHub pre-release tag `vX.X.X-discord`, pushes current branch.
- `Release_CurseForge.command` ‚Äî plain zip + GitHub stable release + CurseForge upload, pushes `main`.
- `Release_Patreon.command` ‚Äî plain handoff zip only, no git ops, for direct distribution.
- `Package_Addon.command` ‚Äî plain unencoded zip only, no git ops, for local testing.

## Locale
- `Locale.lua` is the English fallback. The 10 community locale files live in `Locale/` (deDE frFR esES esMX ruRU zhCN zhTW koKR itIT ptBR).
- Do not add new user-facing strings directly in feature files ‚Äî add to `Locale.lua` first.
